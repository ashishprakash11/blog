How to Publish

1. Fill in GitHub token + repo + branch above and save.
2. When you click Save & Publish on any article, this admin panel pushes data.json directly to your GitHub repo.
3. GitHub Pages automatically rebuilds in ~30 seconds.
4. Your live site reads data.json and renders the new article instantly.

No coding, no command line needed.
